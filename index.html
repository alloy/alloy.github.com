jewelry_portfolio 0.1.0

Imagine writing an erb template once and use rake release to release your gem with Jeweler as usual, but in addition re-generate and push the index.html for your GitHub pages. If that sounds good to you, you're in luck. Because that's exactly what this add-on for Jeweler[http://github.com/technicalpickles/jeweler/tree/master] does.

Install: $ sudo gem install alloy-jewelry_portfolio -s http://gems.github.com

Clone: $ git clone git://github.com/alloy/jewelry_portfolio.git
